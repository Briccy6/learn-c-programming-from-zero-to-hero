# Phase 2: Control Flow in C

This folder contains examples of decision-making and loops in C programming.

## Topics Covered
- if / else if / else statements
- switch / case statements
- Loops: for, while, do-while
- User input and simple calculations

## Files Explained

### `if_else.c`
Determine a grade based on input marks using if-else logic.

### `switch_case.c`
Simple calculator that performs operations based on user choice.

### `loops.c`
Demonstrates for, while, and do-while loops with multiplication table, countdown, and user input.

## How to Compile and Run

```bash
gcc if_else.c -o if_else
./if_else

gcc switch_case.c -o switch_case
./switch_case

gcc loops.c -o loops
./loops
