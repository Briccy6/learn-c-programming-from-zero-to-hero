# Phase 1: C Programming Basics

This folder covers the absolute fundamentals of C programming.

## Topics Covered
- Structure of a C program
- Printing output using printf()
- Variables and data types
- Size of data types in memory

## Files Explained

### `hello_world.c`
Introduces the basic structure of a C program and printing output.

### `variables.c`
Demonstrates variable declaration, initialization, and formatted output.

### `data_types.c`
Explains common C data types and their memory sizes using sizeof().

## How to Compile and Run

```bash
gcc hello_world.c -o hello
./hello
