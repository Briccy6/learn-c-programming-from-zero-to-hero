# Phase 3: Functions in C

This folder introduces functions, recursion, and variable scope in C programming.

## Topics Covered
- Function declaration and definition
- Passing arguments and returning values
- Recursion
- Local and global variable scope

## Files Explained

### `basic_functions.c`
Demonstrates simple functions for addition and subtraction.

### `recursion.c`
Uses recursion to calculate factorial of a number.

### `scope.c`
Explains the difference between local and global variables.

## How to Compile and Run

```bash
gcc basic_functions.c -o basic_functions
./basic_functions

gcc recursion.c -o recursion
./recursion

gcc scope.c -o scope
./scope
